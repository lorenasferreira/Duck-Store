# Duck-Store

Duck-Store es una tienda web sencilla enfocada en mostrar productos (patitos u otros artículos). La interfaz está creada principalmente con HTML y CSS, y usa JavaScript para la interacción. Ajusta las secciones y comandos a las herramientas reales del proyecto si usas frameworks o pipelines adicionales.

## Características
- Catálogo de productos (página principal)
- Visualización de producto individual
- Carrito de compras (si está implementado)
- Diseño responsive para dispositivos móviles y escritorio
- Interacciones básicas con JavaScript (añadir al carrito, filtrado/búsqueda, etc.)

## Tecnologías
- HTML
- CSS
- JavaScript

## Instalación y ejecución
Opciones rápidas para ejecutar localmente:

1. Clonar el repositorio
   git clone https://github.com/lorenasferreira/Duck-Store.git
   cd Duck-Store

2. Abrir localmente (sitio estático)
   - Simplemente abrir `index.html` en el navegador
   - O usar un servidor local:
     - Con Live Server (VS Code): “Live Server”


## Estructura del proyecto
- index.html — entrada principal
- styles/ — estilos
- scripts/ — scripts
- assets/img — imágenes y assets
- README.md — este archivo


## Uso
- Navega por la página principal para ver el catálogo.
- Haz clic en un producto para ver detalles.
- Usa los controles para añadir/quitar productos del carrito.
- En proyectos sin backend, los datos pueden almacenarse en archivos JSON o en localStorage.
